# my-blog-app

A simple blog application where users can create, read, update, and delete blog posts. Users can also comment on posts. Features include user authentication, post management, and comment system.

## Project Structure

This project consists of two main parts:

- **backend/**: Laravel 11 API backend
- **frontend/**: Nuxt 3 frontend application

## Quick Start

### Backend Setup

```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

See `backend/README.md` for detailed setup instructions.

### Frontend Setup

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

See `frontend/README.md` for detailed setup instructions.

## Features

- Laravel 11 backend with RESTful API
- Nuxt 3 frontend with Vue 3 Composition API
- TypeScript for type safety
- Tailwind CSS for styling
- Laravel Sanctum for authentication
- Pinia for state management
